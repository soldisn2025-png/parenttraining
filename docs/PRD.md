# PRD: Walktogether Parent Training App

## Goal
Help Kelly create short weekly parent practice assignments from a 45-video ABA parent-training curriculum.

## Users
- Kelly: single admin clinician.
- Parents: receive no-login assignment links and complete simple weekly checkboxes.

## Core Features
- Deterministic video matching from staff prompt.
- AI-written rationale and weekly guidance only.
- Admin override, finalize, send, reopen, and progress review.
- Parent assignment page with watched/tried/need-help checkboxes.

## MVP Exclusions
- No parent accounts.
- No child records or PHI.
- No multi-admin roles.
- No SMS delivery.
