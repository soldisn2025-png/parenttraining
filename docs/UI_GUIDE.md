# UI Guide

## Brand
Walktogether should feel warm, calm, practical, and family-centered. The logo uses a hand/path/autism-color motif without copying the reference photo.

## Admin UX
Admin screens can be dense but should keep actions obvious: generate, review, revise, approve, send, follow up.

## Parent UX
Parent pages must be mobile-first and readable by a 5th grader. Use short sentences, large tap targets, and one clear action at a time.

Do not show module codes, scores, clinical labels, or technical errors to parents.
